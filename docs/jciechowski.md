# Agregacje
----------------------------
Wykorzystałem dane opisujące trzęsienia ziemi z ostatnich 7 dni [U.S. Geological Survey](http://earthquake.usgs.gov/earthquakes/catalogs/eqs7day-M1.txt)

## Suma trzęsien ziemi w danych regionach

```
collection.aggregate([{
		$group: {
			_id: "$Region",
			total_earthquakes: {
				$sum: 1
			},
		}
	}, {
		$sort: {
			total_earthquakes: -1
		}
	}, ],
```

## Średnia siła trzęsień w regionie

```
collection.aggregate([{
		$group: {
			_id: {
				region: "$Region",
				lat: "$Lat"
			},
			magnitude: {
				$sum: "$Magnitude"
			}
		}
	}, {
		$group: {
			_id: "$_id.region",
			avgMagnitude: {
				$avg: "$magnitude"
			}
		}
	}, {
		$sort: {
			avgMagnitude: -1
		}
	}],
```

## Najsilniejsze oraz najsłabsze trzęsienie w regionie
```
collection.aggregate([{
		$group: {
			_id: {
				Region: "$Region",
				Lon: "$Lon"
			},
			Magnitude: {
				$sum: "$Magnitude"
			}
		}
	}, {
		$sort: {
			Magnitude: 1
		}
	}, {
		$group: {
			_id: "$_id.Region",
			biggestMagnitude: {
				$last: "$Magnitude"
			},
			smallestMagnitude: {
				$first: "$Magnitude"
			}
		}
	}, {
		$sort: {
			biggestMagnitude: -1
		}
	}]
```
